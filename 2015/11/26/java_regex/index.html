Java中使用正则表达式

发表于 2015-11-26 更新于 2018-06-11

在Java中使用正则表达式非常方便，使用
Pattern与Matcher两个类就可以满足需求。

  • Pattern 用来存储所使用的正则表达式
  • Matcher 用来对目标字符串执行正则表达式的匹配
